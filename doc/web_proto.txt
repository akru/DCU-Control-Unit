DCU2 :: The main web - protocol:

  Request stucture:
	[key] [request]

	keys:
		server - server request
		client - client request(be proxed to client module)
	
	server requests:
		login - requet login & password rows, return {auth:true} if success and user "name" row
		logout - return {logout:true} if success 
		clist - return html template list clients
		cinfo - return html template info about client
		cview - return client window content

	clist item:
		name         - client name(string)
		module       - module name(string)
		last_connect - last connect time in seconds(int)
		free         - client is free(bool)
	
	cinfo object:
		description  - small description from module(string)
		version      - client version(string)

	cview object:
		html         - html content(string)
		buttons      - list of smart buttons in bottom(associative array)
